transitiontrees

Predictive pathway discovery in categorical sequence data.

transitiontrees fits variable-depth prediction suffix trees and exposes a pathway-centric API for ranking trajectories, scoring predictions, and quantifying pathway uncertainty under resampling. Validated to machine precision against the archived PST 0.94.1 (Gabadinho & Ritschard 2013).

Full walkthrough: fitting, the pathway-centric API (common_pathways, divergent_pathways, sharp_pathways, path_dependence), pruning, bootstrap stability and informativeness, predictive evaluation, tree introspection, two-tree comparison. 13 sections, on Nestimate::trajectories.

bootstrap_pathways() — Demonstration

Focused demo of the bootstrap design. Validates the test against three Markov regimes (strong, weak, random) simulated with Saqrlab::simulate_sequences(), then applies it to the trajectories dataset. Includes the Monte Carlo confusion-matrix evaluation across 600 simulated datasets.

Methodological provenance

The bootstrap framework extends Saqr, Tikka & López-Pernas (2025), Transition Network Analysis (the tna package) from edges to variable-depth pathways. Pathway-level constructs — stability_rate, informative_rate, flip_consistency — are transitiontrees-specific extensions of the edge-stability framing.
